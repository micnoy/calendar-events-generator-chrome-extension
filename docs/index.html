What It Does

Everything you need to quickly create calendar events

🗣️

Natural Language

Type dates like "next Tuesday at 3pm EST" or "tomorrow at noon" — AI handles the parsing.

📆

Date Picker

Prefer precision? Use the built-in date/time picker with timezone selection. No API call needed.

🔗

Google Calendar

Generates a pre-filled Google Calendar link and opens it instantly. One click to save.

🔒

Encrypted Storage

Your API key is encrypted with AES-256-GCM before being stored. Never saved as plaintext.

Quick Setup

Up and running in under a minute

1
Clone the repo from GitHub
2
Go to chrome://extensions and enable Developer mode
3
Click Load unpacked and select the project folder
4
Click the extension icon → Settings → enter your OpenAI API key
5
Start creating events! 🎉
